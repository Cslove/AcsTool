# android原生开发学习文档

## 背景

我是一个前端开发者，熟悉nodejs reaact typescript，对android原生开发完全不知道。该目录放置各种有关android原生开发相关的文档，包括kotlin语言，以及原生系统库生态相关的各种学习文档，目录一定要按照android领域划分清楚，文档一定要清晰，明了，充分照顾我的前端背景，并加以实际场景的例子给我教学清楚。

每当我需要让你沉淀一个新的知识点时，一定要充分分析该目录已有的目录文档内容，如果已有相关该知识点内容，但没有覆盖我让你的沉淀的该条知识，只在该文档的相关文档领域知识相同的部分作为补充说明教学，禁止添加重复知识目录或者文档！如果你认为我让你沉淀的相关知识，该目录完全没有，那么根据该知识的领域划分创建或者放在对应的目录下。

领域目录可以嵌套划分，一篇文档教学内容不宜过多，尽量拆分至子目录和文档中。

文档内容尽量保持简洁，不要大段说教，代码示例也要保持简洁，力争简单的示例便能充分说明教学该知识点。
